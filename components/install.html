# 安装

使用 yarn|pnpm|npm 来安装，充分利用 Javascript 包和工具的丰富生态系统。

yarn add @young-datafan/datafan-ui
# 或者 pnpm add @young-datafan/datafan-ui
# 或者 npm i @young-datafan/datafan-ui --save
